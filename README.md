# Project Lakshya

Monorepo for the Lakshya financial health platform.

## Structure

- `apps/web` — web application
- `packages/financial-engine` — financial-health domain logic
- `packages/ui` — shared UI components
- `packages/shared` — shared utilities and types
- `docs` — product, architecture, and delivery documentation

## Getting started

Project tooling and implementation will be added as the applications and packages are initialized.
